# 人生如戏

**说明:**

这个是网易minigame比赛的参赛作品，也是我参加的第一个比赛，项目还有许多需要改进的地方，还请大家指正



该游戏为一款基于Unity开发的休闲模拟类游戏，玩家操控一只手偶模拟一天的生活。



##### 技术亮度:

使用到了Unity的布娃娃系统(Ragdoll)


##### 游戏简介：
玩家将操控一只手偶，模拟表演出一名社畜从早上起床洗漱、通勤、
打卡上班、工位摸鱼、被主管责难、加班回家的枯燥而又充满坎坷
的一天。

##### 游戏玩法：
WASD QE  操控左手
IJKL UO 	 操控右手
GH	 左右移动
